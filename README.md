# Financial Instruments Table

A React + TypeScript application to display a table of financial instruments with sorting and styling capabilities.

## Features

- **Sorting**:
  - **Asset Class**: Custom order (Equities > Macro > Credit).
  - **Price**: Descending order (toggleable).
  - **Ticker**: Alphabetical order.
- **Presentation**:
  - Rows color-coded by Asset Class (Macro: White, Equities: Blue, Credit: Green).
  - Price color-coded (Positive: Blue, Negative: Red).

## Tech Stack

- React
- TypeScript
- Vite

## getting Started

1.  **Install Dependencies**:

    ```bash
    npm install
    ```

2.  **Run Development Server**:

    ```bash
    npm run dev
    ```

    Open [http://localhost:5173](http://localhost:5173) to view the app.

3.  **Build for Production**:
    ```bash
    npm run build
    ```
